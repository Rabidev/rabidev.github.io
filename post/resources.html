For learning HTML

Common Resources

For guided learning I will cautiously recomend Kahn Academy. It is a very good, and very detailed introduction to programming in HTML.

    Pros:
  • Basic, beginner level learning.
  • Detailed content.
  • Broken up into a series of short videos.
  • Includes quizzes, knowledge checks and other interactive elements.
    Cons:
  • Honestly might be too beginner level. Ie. you dont need this detailed of a course to learn the basics.
    Once you start learning you'll realize that it is very easy to pick up and "just do it" immediatly.
  • You probably don't need this much of a foundation in HTML yet. You can return to this course once you need more of a solid understanding of HTML


W3Schools has the full HTML 5 documentation on it. It is the go-to site for any HTML reference. They have text and video based totourials and can answer any HTML question you have. Every Web Programmer has this site bookmarked.

For just trying out HTML and experimenting in it use W3Schools "Try It" editor. It will let you try out HTML without installing any text editors. If you want to dive deeper into HTML you will want to set up a text-editor but for now this site is sufficient.
